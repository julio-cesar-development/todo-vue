It stores the data in Local Storage  
> It has unit tests with jest

## Project setup

```bash
npm install
```

### Compiles and hot-reloads for development

```bash
npm run serve
```

### Compiles and minifies for production

```bash
npm run build
```

### Run your tests

```bash
# run tests
npm run test

# run tests in watch mode
npm run test:tdd
```

### Lints and fixes files

```bash
npm run lint
```
